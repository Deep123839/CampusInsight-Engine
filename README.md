 CampusInsight Engine

An Advanced Academic Performance & Placement Analytics Dashboard built using Python, Pandas, and Streamlit.

---

 Overview

CampusInsight Engine is a data-driven dashboard that analyzes student academic performance and placement statistics using interactive visualizations.

The system converts raw CSV data into meaningful insights including:

- KPI Metrics
- Placement Distribution
- Pass/Fail Analysis
- Topper Identification
- Percentage-based Performance Charts
- Dynamic Filtering

This project demonstrates data analysis, dashboard design, and real-time visualization skills.
